Combat de Pokémon
Endpoint : POST /pokemons/battle

Méthodologie : Le combat compare chaque statistique des deux Pokémon (HP, attaque, défense, attaque spéciale, défense spéciale, vitesse).

Résultat : Le Pokémon ayant le plus grand nombre de statistiques supérieures remporte le duel. En cas d'égalité du nombre de statistiques remportées, le résultat est un match nul.
